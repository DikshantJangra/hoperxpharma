# Schedule H1 Register

**Requirement:**
- Schedule H1 drugs (e.g., certain antibiotics, sleep aids) must be recorded in a separate register.
- Details: Date, Doctor Name, Patient Name, Drug Name, Quantity.
- Retain for 3 years.

**System Automation:**
- Mark product as "Schedule H1" in Inventory.
- System automatically adds sales of these items to the "H1 Register Report".
- Export PDF for Drug Inspector visits.
