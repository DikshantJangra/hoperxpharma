# Audit Logs & Activity Tracking

Access: Settings → Security → Audit Logs.

What is logged?
- Login/Logout.
- Price changes.
- Stock adjustments (manual).
- User creation/deletion.
- Prescription verification.

Exporting:
- Logs can be exported to CSV for external audits or regulatory inspections.
- Logs cannot be deleted or modified by any user (including Owner).
