# Incident Reporting

**Types of Incidents:**
- Dispensing Error (Wrong drug/dose).
- Patient Adverse Reaction.
- Staff Injury.

**Procedure:**
1. Compliance → Incidents → Report New.
2. Describe event factually.
3. List corrective action taken (e.g., "Called patient, swapped medicine").
4. Manager review and sign-off.
