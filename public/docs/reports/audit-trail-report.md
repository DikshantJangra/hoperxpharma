# Audit Trail Report

**Filters:**
- User (Who did it?)
- Action (What did they do? e.g., "Price Change")
- Module (Where? e.g., "Inventory")

**Use Case:**
- Investigating why a product price was changed.
- Checking who deleted a customer profile.
- Verifying stock adjustment reasons.
