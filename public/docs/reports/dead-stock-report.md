# Dead Stock Report

**Definition:**
- Items with 0 sales in the last X days (default 180).

**Actions:**
1. **Return:** Check if supplier accepts returns.
2. **Discount:** Run a clearance sale.
3. **Bundle:** Pair with a fast-moving item.
4. **Dispose:** If expired.

Related: `/docs/inventory/supplier-returns`
