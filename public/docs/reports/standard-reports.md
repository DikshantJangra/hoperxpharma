# Standard Reports Overview

Access via **Reports** in sidebar.

## Key Reports
1. **Sales Summary:** Daily/Monthly sales, profit margins, and top products.
2. **Product Performance:** Fast-moving vs. slow-moving items.
3. **Stock Valuation:** Current value of inventory (FIFO/Weighted Average).
4. **User Activity:** Sales by cashier/pharmacist.

Exporting:
- Click **Export CSV/PDF** on top right of any report for offline analysis.
