# Creating Purchase Orders (PO)

Quick Steps
1. Suppliers → Purchase Orders → Create PO.
2. Select Supplier.
3. Add items (manually or "Add Low Stock Items").
4. Save as Draft or Send (Email/WhatsApp).

Receiving against PO:
- When goods arrive, open PO → click **Receive**.
- Verify quantities and batch details.
- System closes PO automatically if fully received.
