# Glossary

- **SKU:** Stock Keeping Unit. Unique code for a product.
- **Batch:** A specific production run of a medicine with a shared expiry date.
- **HSN:** Harmonized System of Nomenclature. Tax code for GST.
- **POS:** Point of Sale. The checkout screen.
- **PO:** Purchase Order. Request to supplier for goods.
- **GRN:** Goods Received Note. Proof of stock receipt.
