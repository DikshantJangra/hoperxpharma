# Receive Stock

Steps
1. Orders → Open PO → Click **Receive**.
2. Scan or select incoming SKUs.
3. Enter batch number, expiry, and quantity received.
4. Save to finalize stock addition.

If you receive without PO: Inventory → Stock Adjustment → Add stock with batch details.

Related: `/docs/suppliers/create-po`, `/docs/inventory/batches`
