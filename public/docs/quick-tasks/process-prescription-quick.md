# Process a Verified Prescription

1. Operations → Prescriptions → Ready for Fulfillment.
2. Select prescription, print label → click **Start Filling**.
3. Scan each product against the prescription; system checks stock & batch.
4. Click **Mark Dispensed** and print the final receipt.
