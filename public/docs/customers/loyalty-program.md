# Loyalty Program Configuration

Quick Steps
1. Settings → Loyalty.
2. Enable Loyalty Program.
3. Set earn rate (e.g., 1 point per ₹100 spent).
4. Set redemption value (e.g., 1 point = ₹1).

Redemption at POS:
- Add customer to cart.
- If points available, "Redeem Points" button appears.
- Click to apply discount.

Tiers:
- Define Silver/Gold/Platinum based on annual spend.
- Set multipliers (Gold earns 1.5x points).
