# Store Configuration & Preferences

Settings → Store → General.

Key settings:
- **Currency Symbol:** ₹ / $ / €
- **Time Zone:** Affects report timestamps.
- **Round Off:** Auto-round total to nearest 0.50 or 1.00.
- **Negative Stock:** Allow/Disallow selling items with 0 stock.

Print Settings:
- Auto-print receipt after sale: ON/OFF.
- Show savings on receipt: ON/OFF.
