# User Management & Roles

Quick Steps
1. Settings → Users.
2. Invite User via email.
3. Assign Role.

Standard Roles:
- **Owner:** Full access to everything.
- **Admin:** Full access except subscription billing.
- **Pharmacist:** Can verify prescriptions, manage inventory, sell. Cannot delete audit logs.
- **Cashier:** Can sell and view own sales. No inventory edit access.

Custom Roles:
- You can create granular roles (e.g., "Inventory Manager") in Settings → Roles → Create New.
