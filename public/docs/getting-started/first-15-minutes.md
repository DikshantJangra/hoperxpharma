# First 15 Minutes

Checklist
1. Verify store profile and license.
2. Add one product.
3. Add a cashier user and login.
4. Connect barcode scanner and printer.
5. Run a test sale and print a receipt.

---

## Step-by-step
1. **Add Product**
   - Inventory → Add Product.
   - Name: `Test Paracetamol`
   - SKU: `TEST-PARA-01`
   - Barcode: `8900000000000`
   - Price: ₹50, MRP ₹50
   - Save.

2. **Open POS and scan**
   - Billing → POS → New Sale.
   - Scan barcode (or search `Test Paracetamol`).
   - Click **Pay** → choose Cash → Confirm.
   - Receipt prints — confirm logo and GST details.

3. **Check Reports**
   - Reports → Sales Today → verify sale appears.

If something fails: check hardware, user permissions, and tax settings under Settings → Taxes.
