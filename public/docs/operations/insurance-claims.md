# Insurance Claims Processing

**Prerequisites:**
- Patient Insurance Card details in Profile.
- TPA Portal Login.

**Workflow:**
1. **Pre-Auth:** Upload Rx and Estimate to TPA portal.
2. **Approval:** Receive Pre-Auth Code.
3. **Billing:** Select Payment Mode "Insurance". Enter Pre-Auth Code.
4. **Documents:** Print Invoice, attach original Rx, get patient signature.
5. **Submission:** Courier documents to TPA for reimbursement.
