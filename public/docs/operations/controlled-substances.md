# Controlled Substances (Narcotics)

**Rules:**
- Cannot be sold without valid physical Rx (no WhatsApp/Email).
- Rx must be retained for 2 years.
- Double-lock storage recommended.

**System Controls:**
- **Hard Stop:** POS prevents sale if "Rx Verified" is not checked.
- **Quantity Limits:** Cannot exceed prescribed amount.
- **Register:** Auto-updates the "Narcotics Register" report.

Related: `/docs/compliance/narcotics-register`
