# Compounding Workflow

**Scenario:** Dermatologist prescribes a custom cream mix.

**Steps:**
1. **Create Compound Product:** Inventory → Add Product → Type "Compound".
2. **Define Recipe:** Add ingredients (e.g., Base Cream 50g + Salicylic Acid 2g).
3. **Batch Creation:** When you make a batch, system deducts raw materials and adds 1 unit of Compound.
4. **Labeling:** Print label with "Compounded Preparation", Expiry (usually short), and Ingredients.
