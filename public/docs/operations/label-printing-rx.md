# Printing Prescription Labels

**Why?**
- Helps patients remember dosage (e.g., "1-0-1 After Food").
- Professional look.

**How to:**
1. In POS/Dispense screen, click "Print Labels".
2. Select printer (Label Printer).
3. System prints one sticker per medicine with:
   - Patient Name
   - Drug Name
   - Dosage Instructions (translated to local language if configured).
   - Expiry Date.

Related: `/docs/quick-tasks/print-label`
