# Doctor Database Management

**Why maintain this?**
- Faster Rx entry (auto-complete doctor name).
- Track "Sales by Doctor" to identify top prescribers.
- Verify registration numbers for compliance.

**Adding a Doctor:**
- Operations → Doctors → Add New.
- Required: Name, Reg Number, Specialty.
- Optional: Clinic Address, Phone (for clarifications).
