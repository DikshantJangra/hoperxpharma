# Patient Counseling Points

**Auxiliary Labels:**
- System auto-suggests labels based on drug class.
- Examples:
  - "May cause drowsiness" (Antihistamines)
  - "Take on empty stomach" (Thyroxine)
  - "Avoid sunlight" (Some antibiotics)

**Printing:**
- These print on the bottom of the prescription label or as a separate "Patient Education Sheet".
