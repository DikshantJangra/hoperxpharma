# Prescription Validity Rules

Settings → Prescriptions → Validity.

**Defaults:**
- **Antibiotics:** 3 days from issue.
- **Chronic Meds:** 6 months or 1 year.
- **Narcotics:** 1 time use only.

**Enforcement:**
- System warns "Rx Expired" if you try to refill an old prescription.
- Pharmacist can override with "Consulted Doctor" note.
