# Returns & Credit Notes

Quick Steps
1. Billing → Transactions → Select sale → Refund.
2. Select items to return, choose refund method.
3. System issues Credit Note if invoice-based.

Notes:
- For GST compliance, return against original Invoice/Invoice number must be referenced in Credit Note.
- For cash refunds: confirm cashier and reason for audit.

Related: `/docs/billing/pos-sale`, `/docs/compliance/audit-logs`
