# Customizing Receipt & Invoice Templates

Quick Steps
1. Settings → Billing → Templates.
2. Upload logo, set footer text, toggle tax details.
3. Save and test print.

Template fields:
- Header: Logo, Store name, address, GSTIN.
- Body: Line item details (name, qty, price, tax).
- Footer: Terms (return policy), signature placeholder, contact.

E-invoicing:
- If enabled, tax invoice format meets IRP (India) requirements; configure in Settings → e-Invoicing.
