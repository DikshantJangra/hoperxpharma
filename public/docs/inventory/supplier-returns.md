# Supplier Returns (Debit Notes)

Quick Steps
1. Inventory → Returns → New Vendor Return.
2. Select Supplier.
3. Add items (scan or select from "Expiring Soon" list).
4. Save & Print "Debit Note".

Financial Impact:
- Creates a Debit Note in accounting.
- You can adjust this amount against future payments to the supplier.

Related: `/docs/inventory/batches`, `/docs/billing/credit-notes`
