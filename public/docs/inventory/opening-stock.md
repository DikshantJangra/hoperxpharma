# Opening Stock Import

1. Download Template: Inventory → Import → Download CSV Template.
2. Fill Columns: Name, SKU, Batch, Expiry, Cost, MRP, Qty.
3. Upload CSV.
4. Review validation errors (if any).
5. Confirm Import.

Tip: Do this store-section by store-section to avoid overwhelming data errors.

Related: `/docs/inventory/bulk-import-export`
