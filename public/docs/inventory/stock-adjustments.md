# Stock Adjustments & Corrections

Quick Steps
1. Inventory → Adjustments → New Adjustment.
2. Select Reason (Breakage/Theft/Data Entry Error).
3. Scan product and enter quantity to remove (negative) or add (positive).
4. Save.

Audit Trail:
- All adjustments are logged with the user ID and reason.
- Large adjustments may require Manager approval (configure in Settings).
