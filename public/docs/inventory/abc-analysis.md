# ABC Analysis & Classification

System automatically classifies items based on consumption value:
- **Class A:** Top 20% items contributing 80% revenue. (Tight control, low safety stock).
- **Class B:** Next 30% items contributing 15% revenue. (Moderate control).
- **Class C:** Bottom 50% items contributing 5% revenue. (Bulk order, high safety stock).

View Report: Reports → Inventory → ABC Analysis.
