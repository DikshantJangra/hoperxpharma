# Bulk Import & Export Tools

Use Case: Updating MRP for 500 items at once.

Steps:
1. Inventory → Export → Products.
2. Open CSV in Excel.
3. Update "MRP" column.
4. Inventory → Import → Update Existing Products.
5. Upload CSV.

Warning: Always backup (Export) before importing bulk updates.
