# Inter-Store Stock Transfers

Quick Steps
1. Inventory → Transfers → New Transfer.
2. Select Source Store and Destination Store.
3. Add items and quantities.
4. Status: "In Transit".

Receiving Store:
- Opens Transfer → Clicks "Receive".
- Verifies goods.
- Stock deducts from Source and adds to Destination.
