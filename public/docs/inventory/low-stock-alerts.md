# Low Stock Alerts Configuration

1. Settings → Inventory → Notifications.
2. Enable "Low Stock Email Alert".
3. Set frequency (Daily/Weekly).
4. Add recipient emails.

Dashboard:
- "Low Stock" widget on dashboard updates in real-time.
