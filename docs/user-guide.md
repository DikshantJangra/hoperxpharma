# HopeRxPharma - Patient Management User Guide

## Overview
The new Patient Management module allows pharmacists to manage patient records, prescriptions, refills, and consents efficiently.

## Key Features

### 1. Patient Profile
- **Access**: Click on any patient in the Patient List.
- **Summary**: View key stats, loyalty points, and recent activity.
- **Tabs**: Navigate between Prescriptions, Sales, Consents, and History.

### 2. Processing Refills
- **How to**: Click "Refill" on the Patient Profile or "Refills Due" on the Dashboard.
- **Workflow**:
  1. Select the prescriptions to refill.
  2. Adjust quantities if needed.
  3. Click "Process Refill".
- **Note**: This will automatically create a sale draft and update inventory.

### 3. Uploading Prescriptions
- **How to**: Click "Upload Rx" on the Patient Profile.
- **Features**:
  - Upload image or PDF.
  - Automatic text extraction (OCR) for medicine names.
  - Save to patient record.

### 4. Managing Consents (DPDPA 2023)
- **View**: Go to the "Consents" tab on the Patient Profile.
- **Withdraw**: Click the "Withdraw" button on an active consent to revoke it.
- **Note**: Messaging (WhatsApp/SMS) is blocked if consent is withdrawn.

### 5. Merging Duplicate Patients
- **When**: If you find two records for the same patient (e.g. same phone number).
- **How to**: 
  1. Click "Merge" on the Patient Profile.
  2. Search for the duplicate record.
  3. Select which details to keep.
  4. Confirm merge.
- **Warning**: This action cannot be undone.

### 6. Offline Mode
- **Status**: The app automatically detects if you lose internet connection.
- **Actions**: You can still view cached patients and queue actions (like refills).
- **Sync**: Changes will automatically sync when you come back online.

## Keyboard Shortcuts
- `F2`: Focus Search
- `F3`: Quick Add Patient
- `F12`: Create Sale (from Profile)
- `Esc`: Close Modal

## Support
For issues, contact the IT support team or check the "History" tab for audit logs.
