# Print Product Label

Steps
1. Inventory → Products → Select product.
2. Click **Print Label**.
3. Choose label size and quantity.
4. Click **Print**.

Tip: Use standard 58x40 mm labels for shelf tags and 25x12 mm for box stickers.

Troubleshooting:
- Label not aligned: check printer margin settings or run printer calibration.
