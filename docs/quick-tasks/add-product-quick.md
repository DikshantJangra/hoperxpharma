# Quick Add Product

Quick Steps
1. Inventory â†’ Add Product.
2. Fill `Name`, `SKU`, `Barcode`, `MRP`.
3. (Optional) Tick `Batch tracked`.
4. Save.

Fields explained:
- SKU: unique internal code.
- Barcode: EAN/UPC/GS1 string scanned at POS.
- MRP: maximum retail price (printed on product).

Troubleshooting:
- Duplicate SKU error: change SKU to unique value.
- Barcode scanner doesn't register: test scanner in a text editor.
